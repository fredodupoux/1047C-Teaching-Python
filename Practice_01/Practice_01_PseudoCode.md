# OIP

### OUTPUT
- Workers Name
- Gross Pay

### INPUT
- Workers name
- Hours worked
- Hourly rate

### PROCESS
- Compute gross pay: Hours worked \(*\) Hourly rate
- Round Gross pay result

### PSEUDO CODE

```plaintext
Input Workers Name
Input Hours Worked
Input Hourly Rate
Calculate Gross Pay (Hours Worked * Pay Rate)
Round Gross Pay
Display Workers Name and Gross Pay
```
